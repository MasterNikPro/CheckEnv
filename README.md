# 🔍 CheckEnv – Tiny DevOps Tool for Environment Validation

> **"Because forgetting your API key shouldn’t break production."**

---

## 🚀 Overview

**CheckEnv** is a lightweight command-line Python tool built for developers and DevOps engineers who want a fast way to verify if essential environment variables are set before running or deploying applications.

It’s small, simple, and perfect for CI/CD pipelines or local dev sanity checks.

---

## 🛠️ Features

✅ Validates system environment variables  
✅ Supports both **default** and **custom** variable checks  
✅ Clear terminal output for missing variables  
✅ Zero dependencies – just Python  
✅ Easy to plug into shell scripts or CI/CD jobs

---

## 📦 Installation

No dependencies required!  
Just clone the repo and run with Python 3.
